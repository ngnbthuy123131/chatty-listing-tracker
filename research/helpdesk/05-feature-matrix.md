## Table-Stakes vs. Differentiators

### Table-Stakes Features (Must-Have for Any Helpdesk)
Every helpdesk entering this market MUST have these on day 1:

1. **Shared inbox** тАФ Unified view of all conversations
2. **Email integration** тАФ Connect store email addresses
3. **Ticketing** тАФ Create, assign, track, resolve
4. **Macros/templates** тАФ Saved replies for common questions
5. **Tags & categorization** тАФ Organize and filter tickets
6. **Internal notes** тАФ Team collaboration on tickets
7. **Basic automation rules** тАФ Auto-assign, auto-tag, auto-close
8. **Customer profile sidebar** тАФ Show Shopify order data alongside ticket
9. **Basic reporting** тАФ Volume, response time, resolution time
10. **Collision detection** тАФ Prevent two agents replying simultaneously
11. **Knowledge base** тАФ Self-service FAQ articles
12. **Mobile app** тАФ Agents need to respond on the go

### Differentiators (What Wins Market Share)

#### Tier 1 Differentiators тАФ High Impact
1. **Deep Shopify order actions** тАФ Edit, refund, cancel orders WITHOUT leaving the helpdesk. This is Gorgias's moat. Any serious competitor must match it.
2. **Revenue attribution** тАФ Prove support drives revenue. Only Gorgias does this well.
3. **AI-powered auto-resolution** тАФ Handle WISMO, returns, FAQs automatically. The future of helpdesk.
4. **Predictable pricing** тАФ NOT per-ticket. This is Gorgias's weakness and Chatty's biggest opportunity.

#### Tier 2 Differentiators тАФ Competitive Edge
5. **Omnichannel beyond email** тАФ Social DMs (FB, IG, TikTok), WhatsApp, SMS from one inbox
6. **Self-service portal** тАФ Let customers track orders, initiate returns themselves
7. **Ecommerce integrations** тАФ Klaviyo, Recharge, Loop Returns, Yotpo etc.
8. **CSAT surveys** тАФ Measure satisfaction post-resolution
9. **SLA management** тАФ Set and track response/resolution time targets
10. **Multi-brand support** тАФ Manage multiple stores from one account

#### Tier 3 Differentiators тАФ Nice-to-Have
11. **Live visitor view** тАФ See who's on your site in real-time
12. **Video calls** тАФ In-chat video support
13. **Screen sharing** тАФ See customer's screen
14. **Custom roles & permissions** тАФ Enterprise-grade access control
15. **Shopify Flow integration** тАФ Trigger automations across the Shopify ecosystem

## Chatty's Recommended Feature Priorities

### Phase 1 тАФ MVP (Match Table Stakes)
- Shared inbox with email integration
- Shopify customer/order sidebar (view orders, customer history)
- Basic ticketing (assign, tag, status, priority)
- Macros/saved replies
- Internal notes
- Collision detection
- Basic automation rules (auto-assign, auto-tag)
- Knowledge base / FAQ
- Basic reporting (volume, response time)
- **Free plan with generous limits**

### Phase 2 тАФ Compete (Match Gorgias Core)
- Order actions from helpdesk (refund, cancel, edit)
- Social media channels (FB, IG DMs)
- CSAT surveys
- Advanced automation rules
- Revenue attribution
- WhatsApp integration
- SLA management

### Phase 3 тАФ Differentiate
- AI auto-resolution (WISMO, returns, FAQ) at competitive/included pricing
- Self-service customer portal
- Shopify Flow integration
- Multi-brand support
- Ecommerce app integrations (Klaviyo, Recharge, Loop)
- Predictable flat-rate pricing as the core value proposition vs. Gorgias
