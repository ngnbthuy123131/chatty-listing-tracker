## Table Stakes vs Differentiators

### Table Stakes (Must-Have for Any Livechat App)
These features are expected by every merchant. Missing any = instant disqualification:

1. **Real-time chat widget** вЂ” customizable colors, position
2. **Mobile app** вЂ” agents must respond from phone
3. **Offline messages** вЂ” capture leads when agents unavailable
4. **Canned responses** вЂ” efficiency for common questions
5. **Basic customization** вЂ” brand colors, agent avatars, welcome message
6. **Shopify integration** вЂ” at minimum customer info, order lookup
7. **Notification system** вЂ” sound, desktop, push notifications
8. **Chat history** вЂ” searchable conversation archive

### Differentiators (Create Competitive Advantage)
Features that justify a paid subscription over Shopify Inbox:

1. **Real-time visitor list** вЂ” see who's browsing, what they're viewing
2. **Proactive chat triggers** вЂ” auto-message based on behavior (time on page, exit intent, cart value)
3. **Typing insights** вЂ” see what visitor types before sending (Chatra's killer feature)
4. **Multi-channel inbox** вЂ” Messenger, Instagram, WhatsApp, email in one dashboard
5. **Cart preview + product recommendations** вЂ” agent sees cart, can suggest products
6. **Advanced analytics** вЂ” response times, resolution rates, agent performance
7. **Chat routing + departments** вЂ” assign chats to right agent/team
8. **Co-browsing** вЂ” see visitor's screen (Crisp's MagicBrowse)
9. **Video/voice chat** вЂ” escalate from text to call
10. **AI-powered suggestions** вЂ” smart replies, auto-categorization

### Premium Features (Justify $30+/mo)
What separates premium from commodity:

1. **AI chatbot / auto-responder** вЂ” handle common queries without human
2. **Advanced automation workflows** вЂ” triggers, conditions, actions
3. **Custom branding** вЂ” remove third-party logo
4. **SSO / enterprise security** вЂ” for larger teams
5. **API access** вЂ” custom integrations
6. **Staffing predictions** вЂ” when to have agents online
7. **Revenue attribution** вЂ” track sales influenced by chat

## What Makes a Livechat "Premium" vs "Commodity"

### Commodity Livechat (Race to Bottom)
- Basic chat widget on website
- Simple message exchange
- Email fallback when offline
- Basic agent dashboard
- **Value perception: $0-10/mo** (Shopify Inbox, Tawk.to set this expectation)

### Premium Livechat (Defensible Value)
- **Intelligence layer**: Know WHO you're talking to (cart contents, browse history, order history, customer lifetime value)
- **Proactive engagement**: Don't wait вЂ” trigger chats based on behavior patterns
- **Multi-channel unified inbox**: One dashboard for website chat + Messenger + Instagram + WhatsApp + email
- **Team efficiency**: Routing, departments, canned responses, macros, internal notes
- **Measurable ROI**: Analytics showing revenue influenced, conversion lift, CSAT scores
- **AI augmentation**: Smart suggestions for agents, auto-responses for simple queries
- **Value perception: $20-50/mo**

### The Commodity Trap
Pure livechat is becoming a commodity. The feature set is well-defined, easily replicated, and Shopify gives it away free. Winners in this space will differentiate through:

1. **Shopify-specific depth** вЂ” not just "works with Shopify" but deeply embedded (product recommendations in-chat, discount code generation, order status lookup)
2. **AI that actually helps** вЂ” not gimmicky, but genuinely reducing agent workload
3. **Ease of use** вЂ” setup in <5 minutes, zero learning curve
4. **Speed** вЂ” fast widget load, instant message delivery
5. **Design quality** вЂ” beautiful, modern widget that merchants are proud to show

## Opportunity Gaps for Chatty

Based on the matrix analysis, underserved areas include:

1. **Deep Shopify integration at low cost** вЂ” Tidio does this but is increasingly expensive. Shopify Inbox does it free but with basic features. There's room for a mid-tier option.
2. **Flat pricing model** вЂ” Only Crisp does this well. Most competitors still use per-agent pricing.
3. **Multi-channel for SMBs** вЂ” WhatsApp/Instagram/Messenger unified inbox at <$25/mo is rare.
4. **Proactive sales features** вЂ” Cart-based triggers, product recommendations, discount offers during chat. Not just support, but sales conversion.
5. **Beautiful, lightweight widget** вЂ” Many widgets are heavy (slow page load) or ugly. A fast, gorgeous widget is a real differentiator for design-conscious Shopify merchants.
