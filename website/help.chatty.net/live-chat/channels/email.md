# Email Integration

> Source: https://help.chatty.net/live-chat/channels/email

## Availability

This feature is available for all users.

## Supported Email Providers

- Gmail
- Outlook
- Zoho
- Hostinger
- Amazon SES
- Other providers (contact support if unlisted)

## Gmail Setup

1. Access Gmail settings and navigate to "Forward and POP/IMAP"
2. Add the forwarding address from the Chatty instruction panel
3. Confirm the forwarding by accessing a verification link
4. Enable the forwarding option to copy incoming mail
5. Send a verification email for final confirmation

## Outlook Configuration

1. Sign in via web browser (with guidance for app users)
2. Access Settings and Mail forwarding options
3. Enable forwarding and enter the provided address
4. Send a verification email

**Important Note:** Organizations may restrict external forwarding. The error message "550 5.7.520 Access denied, Your organization does not allow external forwarding" requires IT administrator approval to resolve.

## Zoho Email Process

1. Access Settings and Mail accounts
2. Go to Forwards and add the Chatty forwarding address
3. Click Verify and obtain a confirmation code
4. Paste the code in Chatty and confirm
5. Send a final verification email

## Hostinger Configuration

1. Sign in and click Manage
2. Create a forwarder in the Forwarders section
3. Enter the Chatty forwarding address
4. Send a verification email

## Other Providers

For unlisted providers, search settings for forwarding options and contact support for specific assistance.

## FAQs

**Multiple Email Providers:** Chatty works with most providers including Gmail, Outlook, and custom domains using the same verification approach.

**Alias Email Verification:** The verification link is sent to your primary (main) email address, not to the alias itself. Verify your primary email first, then add aliases without requiring additional inbox verification.

**Primary Email Removal:** You can verify the primary email, add aliases, then remove the primary from Channels while keeping aliases active.
