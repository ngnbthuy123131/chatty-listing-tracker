# Translation

> Source: https://help.chatty.net/others/translation

## Availability

This feature is available for all users.

## Overview

The Translation feature enables automatic content translation within chatboxes to users' preferred languages, eliminating manual translation effort for FAQ labels.

## How Translation Works

Chatboxes display in the default language initially for all visitors. When additional languages are published, they become accessible to users whose website language matches those options.

### Website Language Setup

Before publishing a new language, ensure your website supports it:

1. Access Shopify Admin
2. Navigate to **Settings** → **Language**
3. Configure your desired languages

## Supported Languages

Chatty currently supports 19 languages:

English, Spanish, French, Portuguese, German, Dutch, Italian, Turkish, Vietnamese, Hindi, Urdu, Chinese (Simplified), Chinese (Traditional), Arabic, Malay, Japanese, Greek, Romanian, and Norwegian.

For unsupported languages, select **Custom** as an alternative.

## Default Language

The default language displays to all users and is set to English upon initial app setup. If a user's language isn't available in your translations, the chatbox reverts to the default language.

**Note:** Users who installed the app before April 23, 2024, retain their previously set primary language as default.

### Changing Default Language

Only published languages can be set as default:

1. Go to Translations
2. In the published language list, click the edit icon
3. Select **Change default**
4. Click **Save**

## Publishing Languages

### Publishing Process

1. Click **Add language**
2. The new language appears under "Unpublished language" with auto-translated content
3. Click **Edit** to modify translations (optional)
4. For all content areas, click **Auto-translate** or manually edit text
5. In FAQs translation, use the edit icon to modify categories and questions
6. Click **Save** after editing
7. Click the edit icon → **Publish** → confirm **Publish**

Published languages become available to website visitors using those languages.

## Translation Updates

New FAQ categories and questions don't automatically update existing translations. An "Outdated" status appears on languages needing updates. Click **Edit** to refresh translations with new content.
