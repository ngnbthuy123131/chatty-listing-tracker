# Notifications

> Source: https://help.chatty.net/others/notifications

## Overview

Setting notifications for messages helps you take care of all customers. No missing messages.

This feature is accessible to all users and offers three notification types.

## Three Notification Types

1. **Web push notifications** - Browser-based alerts with optional sound
2. **Email notifications** - Message updates sent to email addresses
3. **Mobile notifications** - Requires downloading the Chatty mobile application

## Web Push Notifications Setup

### Step 1: Enable Notifications in App

Navigate to Notifications settings and select "Web" for desired alert scenarios:
- Conversation assigned to you
- New message from unassigned conversations
- New message from conversations assigned to you
- New message from AI assistant conversations
- No replies within 15 minutes from assigned conversations

### Step 2: Activate on Web Browser

Supported browsers: Chrome, Microsoft Edge, Firefox, Opera, and Samsung Internet.

### Step 3: Enable on Device

**Windows:**
1. Settings → System → Notifications (or Notifications & actions)
2. Find your web browser in "Get notifications from these senders"
3. Enable notifications

**MacOS:**
1. Apple menu → Notifications sidebar
2. Locate your web browser in Application Notifications
3. Enable notifications

**Mobile:** Download the Chatty app for mobile message notifications.

## Browser-Specific Instructions

### Chrome
1. In Web push section, click Manage
2. A new window opens; click the settings icon
3. Turn on Notifications
4. Return to Chatty and click Save
5. Click Send test to verify functionality

### Microsoft Edge
1. In Web push section, click Manage
2. Turn on Notifications in the new window
3. Refresh the page and click Save
4. Click Send test to confirm notifications work

### Firefox
> We cannot help you grant permission for Chatty in Shopify embed mode if you use Firefox.

1. Click Set up in web app in top banner
2. In web app, click Enable in banner
3. Click the settings icon in browser popup
4. Select Allow
5. Return to Chatty, refresh, and click Send test

### Opera
1. Turn on Web push notifications
2. Click Allow in popup window
3. Refresh page and click Save
4. Click Send test to verify

## Sound Notifications

By turning sound notification on, your customer will get notifications for new messages with sound from devices.

Configure preferences for receiving sounds:
- Only first message
- All messages

## Email Notifications

Enable email alerts by selecting notification triggers:
- Conversation assigned to you
- New message from unassigned conversations
- New message from assigned conversations
- No replies (for new messages) within 15 minutes from assigned conversations

Click Save and Send test to receive sample notification.

### Managing Multiple Email Addresses
Click Manage in the Email section to add additional recipient addresses.

## Preventing Email Notifications from Spam

### Method A: Create Email Filter
1. Open Chatty email
2. Click three-dot menu next to sender name
3. Select Filter messages like this
4. Click Create filter
5. Choose "Never send to Spam" and "Always mark as important"
6. Click Create filter to save

### Method B: Mark as Important
1. Open Chatty email
2. Click the importance marker icon next to the subject
3. Icon turns yellow when marked important
