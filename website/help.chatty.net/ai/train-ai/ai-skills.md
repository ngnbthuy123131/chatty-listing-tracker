# AI Skills

> Source: https://help.chatty.net/ai/train-ai/ai-skills

## Availability

This feature is available for all users.

## Overview

AI skills represent specialized training modules that enable AI assistants to handle specific customer scenarios effectively. These capabilities allow systems to provide product recommendations, size guidance, support automation, and team handoffs.

### Core Capabilities

When enabled, AI skills permit:
- Product recommendations aligned with customer preferences
- Accurate sizing information
- Automated support for common inquiries
- Seamless escalation to human agents

## Skill Categories

### Shopping Skills
- Smart recommendations
- Size guidance
- Inventory information
- Clarifying follow-up questions

### Customer Support Skills
- Human support requests
- Post-purchase issue resolution
- Order status inquiries

## Benefits

- Increased sales through targeted product displays
- Reduced cart abandonment via sizing confidence
- Automated handling of repetitive support questions

## Setup Instructions

### Smart Recommendations

**Four collection types:**
1. **Bestsellers** - Responds to "What's popular?"
2. **New arrivals** - Responds to "What's new?"
3. **Sales promotions** - Responds to discount inquiries
4. **Special occasions** - Responds to gift-related questions

**Smart Sync Feature:** System automatically selects top 20 products based on 30-day sales performance, updating daily. Users may combine auto-selection with manual curation (maximum 20 products per collection).

**Configuration steps:**
1. Access Smart recommendations
2. Select collection to manage
3. Configure product list and keywords
4. Add or delete products as needed
5. Save and activate

### Size Guide

**Setup process:**
1. Navigate to AI assistant settings
2. Select Configure agents under Agent instructions
3. Click Manage beside size guide
4. Upload image file (JPG/PNG, max 5MB)
5. Designate applicable products
6. Add custom instructions if needed
7. Enable toggle to activate

**Purpose:** Prevents AI from fabricating sizing details by establishing clear instructions for unknown product information.

### Inventory Status

Controls stock-related responses with default smart logic handling:
- In-stock situations
- Out-of-stock conditions
- Backorder or pre-order scenarios

Custom instructions override defaults for specific store requirements.

## Customer Support Skills (Default-Enabled)

Three automated functions activate automatically:

**Transfer** - Facilitates handoff when customers request human assistance

**Refund-Return** - Sends form for post-purchase issues

**Order Tracking** - Retrieves customer information and provides shipment details directly in conversation
