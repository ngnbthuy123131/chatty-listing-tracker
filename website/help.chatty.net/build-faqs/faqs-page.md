# FAQs Page

> Source: https://help.chatty.net/build-faqs/faqs-page

## Availability

This feature is available for all users.

## Overview

An FAQs page displays commonly asked questions with answers, enabling customers to find solutions independently without contacting support.

## Setup Instructions

### Step 1: Access FAQ Settings
Navigate to the FAQs section and select the FAQ page option.

### Step 2: Enable the Feature
Activate the "Display FAQs page" toggle.

### Step 3: Configure URL
Establish the page's web address.

### Step 4: Customize Appearance
- Select your preferred layout style
- Adjust colors for titles, questions, and backgrounds
- **Accessibility Note:** If you customize your text & background color, make sure contrast is good enough for users to read.
- Recommended tool: https://coolors.co/contrast-checker

### Step 5: Set Up Header
Personalize the heading and description text.

### Step 6: Add Contact Option
Configure a "Contact us" element:
- Enable the display toggle
- Customize heading and description
- Define the click behavior action

### Step 7: Advanced Configuration
- Remove Chatty branding (optional)
- Apply custom CSS for advanced customization (requires coding knowledge)

## Adding FAQs to Navigation

### Main Menu Integration
1. Access Shopify admin > Content > Menus
2. Select "Main menu"
3. Click "Add menu item"
4. Enter a label (e.g., "FAQs")
5. Search for "Frequently Asked Questions" or use your custom URL
6. Save the menu
7. Verify in your storefront

### Footer Integration
1. Access Shopify admin > Content > Menus
2. Select "Footer menu"
3. Add the FAQs link
4. Verify in your storefront footer
