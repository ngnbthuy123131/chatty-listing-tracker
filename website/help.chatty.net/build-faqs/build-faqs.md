# Build FAQs

> Source: https://help.chatty.net/build-faqs

## Overview

The Build FAQs section covers essential functionality for creating and managing frequently asked questions.

## Key Features

**Display Options:**
- FAQs can appear as a dedicated page on your Shopify storefront
- Alternatively, FAQs can be embedded as a section within other pages

**Management Capabilities:**
- Add categories to organize questions
- Add individual questions and answers
- Import/export FAQs via CSV

**Performance Tracking:**
Key metrics about your FAQs, like total views, so you can see how users are interacting with them.

## Documentation Structure

- [Add Category](add-category.md) - How to add or remove categories of questions
- [Add Questions](add-questions.md) - How to add or remove new questions and answers
- [FAQs Page](faqs-page.md) - Show a page of FAQs on your Shopify site
- [FAQs Block](faqs-block.md) - Show FAQs as a section on different pages
- [FAQs Analytics](faqs-analytics.md) - View metrics and engagement data
