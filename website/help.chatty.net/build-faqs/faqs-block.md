# FAQs Block

> Source: https://help.chatty.net/build-faqs/faqs-block

## Availability

This feature is available for all users.

## Overview

The FAQs block feature enables merchants to display contextual frequently asked questions across different store pages. This tool allows for custom FAQ blocks with specific questions that appear only on relevant product pages.

## Key Benefits

- Creating multiple FAQ blocks tailored to specific products or collections
- Independent customization of each block's visual presentation
- Precise placement control throughout the store
- Organizing questions by product relevance

## Setup Instructions

1. Navigate to FAQs from the main navigation menu
2. Access the FAQs block section
3. Configure general settings (block name, heading, and subheading content)
4. Select which FAQs to include via the Browse function
5. Choose display conditions (all pages, specific products, or collection pages)
6. Customize visual elements (heading size, alignment, card radius, color)
7. Save your configuration
8. Copy the Block ID
9. Open the theme editor
10. Add the Chatty FAQs block as a section and paste the Block ID
11. Save changes in the theme editor

## Alternative Implementation

Users can also add FAQ blocks using HTML code by accessing FAQs block code, copying it, and inserting it into their website.

## Management Features

The management interface displays all blocks with their associated Block ID, name, FAQ count, display conditions, and status, allowing for editing and deletion as needed.

## FAQs

**Quantity limit:** You can create as many different FAQs blocks as you need for your store.

**Reusability:** Yes, you can use the same questions and answers across different blocks by using same ID block.
