# Add Questions

> Source: https://help.chatty.net/build-faqs/add-questions

## Availability

This feature is available for all users.

## Overview

After establishing a category, users can populate it with FAQ content.

## Adding a New Question

1. Navigate to **FAQs** section
2. Select **Add new**
3. Choose **Add FAQ**
4. Input your question and corresponding answer text
5. Configure the FAQ settings:
   - Choose publication status (Published or Draft)
   - Assign to a category (mandatory field)
   - Enable "Featured question" option to display this item prominently in the initial chatbox view
6. Click **Save**

## Managing Existing FAQs

- **Editing:** Hover over a question and select the edit icon
- **Deleting:** Hover over a question and click the delete icon
- **Reordering:** Drag questions to rearrange their position within categories
- **Featuring:** Star or unstar categories to designate them as featured

## Importing FAQs via CSV

### Step 1: Obtain Sample File
Navigate to FAQs → More actions → Import FAQs → Download sample CSV

### Step 2: Populate Your Data
Complete all required fields in your CSV. The file must contain:
- Question
- Answer
- Category (category name association)
- Published question (TRUE/FALSE)
- Featured question (TRUE/FALSE)

*Maximum file size: 1MB*

### Step 3: Upload and Configure
Upload your CSV file and select import preferences:
- Overwrite existing FAQs with identical questions
- Publish all newly created categories

## Exporting FAQs

Go to FAQs → More actions → Export FAQs → Select desired FAQs → Export

Exported files include 7 fields, adding:
- Published category status
- Featured category status
