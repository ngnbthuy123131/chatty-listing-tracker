# FAQs Analytics

> Source: https://help.chatty.net/build-faqs/faqs-analytics

## Availability

This feature is available for all users.

## Overview

The FAQs analytics feature enables you to monitor total view counts for frequently asked questions. Each expansion of an FAQ answer -- whether on the dedicated FAQs page, within an FAQ block, or through the chatbox -- registers as a view.

## How to Check FAQ Views

### Method 1: FAQs Section
Navigate to the **FAQs** section to observe the total view count for each individual question.

### Method 2: Dashboard
Access the **Dashboard** to:
- Review an overview of FAQ views
- Filter analytics data by specified time ranges

## Key Metrics Tracked

The system records view counts each time users click to expand and view answers across multiple surfaces where FAQs appear.
