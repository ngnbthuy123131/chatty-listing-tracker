# Add Category

> Source: https://help.chatty.net/build-faqs/add-category

## Availability

This feature is available for all users.

## Understanding Categories

Categories function as organizational tools for grouping related questions by topic. The system includes two default categories frequently used by Shopify merchants:
- Order & Shipping
- Exchange & Return

Users have full control to modify existing categories, introduce new ones, and remove categories as needed.

## Steps to Create a New Category

### Step 1: Navigate to FAQs
Access the FAQ management section of your dashboard.

### Step 2: Initiate Category Creation
Select "Add new" → Choose "Add category"

### Step 3: Configure Category Settings

- **Category name:** The title displayed on your FAQ page
- **Icon:** Pick an appropriate visual icon for user comprehension. Additional options available via "View more." Custom icons may be uploaded through the icon upload feature.
- **Position:** Determines category ordering from top to bottom on the FAQ page
- **Status:** Publish immediately or keep as draft
- **Feature category:** Displays the category prominently on the chatbox's first page

> If you enable featured for 2 categories, the one with higher position will be placed first.

### Step 4: Save Changes
Click the Save button to finalize your new category.

## Managing Existing Categories

- **Edit:** Hover and click the edit icon
- **Delete:** Hover and click the delete icon
- **Reorder:** Drag categories to desired positions
- **Feature/Unfeature:** Click the star icon to toggle featured status
