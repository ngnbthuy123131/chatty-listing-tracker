# Contact Button

> Source: https://help.chatty.net/chatbox/contact-button

## Availability

This feature is accessible to all users and functions on Product and Cart pages.

## Overview

The contact button feature enables you to offer immediate support and enhance customer experience.

## Setup Instructions

### Step 1: Access Chatbox Settings
Navigate to your Chatbox configuration area.

### Step 2: Enable the Feature
Select "Turn on" for the Contact button option.

### Step 3: Navigate to Theme Editor
Click "Go to theme" to proceed with adding the button to your site's design.

### Step 4: Position and Add Block
In the theme editor, choose your desired location and click "Add block."

### Step 5: Select Contact Button Component
From the available options, choose "Contact button by Chatty."

### Step 6: Configure Settings
Customize the contact button through the right panel options.

**Important:** You can only set up contact button with active contact methods. Active contact methods are those you set up in chatbox blocks.

## Removal Instructions

To delete the contact button:
1. Open the theme editor
2. Select the Contact button block
3. Click "Remove block"

> If you delete contact method in Chatty, contact button won't be deleted. Go to theme editor and delete that block.
