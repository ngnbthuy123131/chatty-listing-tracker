# Chatbox Settings

> Source: https://help.chatty.net/chatbox/chatbox-settings

## Availability

This feature is available to all users.

## Overview

The chatbox is a website popup featuring sections for order tracking, customer contact, and frequently asked questions.

## Main Customization Categories

### General Settings

Configure basic functionality including:
- Enabling/disabling the chatbox
- Activating chat focus mode
- Customizing header with logo, heading, and description
- Toggling four content blocks: "Contact us," "Order tracking," "FAQs," and "Categories"

The contact section supports eleven communication channels: WhatsApp, Messenger, phone, email, Instagram, Telegram, Skype, Line, Zalo, TikTok, and SMS.

### Appearance Settings

Modify visual elements through:
- Brand color selection from presets or custom options
- Chatbox button configuration (launcher style, icon, size, alignment)
- Layout options (navigation toggle and mobile ratio adjustment)

### Advanced Settings

Access sophisticated features:
- Deep link implementation for targeted navigation
- Device and page targeting (desktop/mobile, all pages or specific ones)
- "Continue as email" functionality allowing conversations to shift from live chat to email
- Custom CSS code for personalized FAQs pages

## Chat Page Configuration

Customize the chat experience through:
- **Welcome message**: Initial greeting when visitors start chatting
- **Conversation starters**: Suggested questions displayed in chat
- **Chat avatar**: Representative image shown during conversations
- **Entry method**: Options for guest chat (form required), anonymous chat, or both
- **Pre-chat form**: Collects visitor email, name, and phone information

## Unread Message Counter

A numerical indicator displays on the chat widget, showing pending unread messages or welcome messages awaiting response.
