# Deep Links

> Source: https://help.chatty.net/chatbox/deep-links

## Availability

This feature is available for all users.

## Overview

Deep links enable creation of specialized URLs that automatically launch the Chatty chatbox when accessed on your Shopify store. These links can be integrated into buttons or hyperlinked text throughout your site.

## Implementation Steps

### Step 1: Access Settings
Navigate to the Chatbox section and select the Advanced option.

### Step 2: Copy Your Deep Link
Copy the deep link for the desired chatbox tab. For example, if you want to open the "Order tracking" tab in the chatbox, copy the deep link for Order tracking.

Different tabs within the chatbox have corresponding deep links available for selection.

### Step 3: Integrate the URL
Append the copied deep link to your website's URL structure.

### Step 4: Add Trigger Element
Add the link to a button or anything on your website to use as a trigger.

## Usage Example

A practical application involves adding a new button to your website's navigation menu that links to the deep link URL. When visitors click this navigation button, the chatbox opens automatically.

This approach allows merchants to strategically place chatbox access points throughout their Shopify stores, improving customer accessibility to support features like order tracking.
