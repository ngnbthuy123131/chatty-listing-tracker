# Re:amaze Full Feature Analysis

## 1. FEATURE LIST THEO PRICING PLAN

### Starter Plan - $59/tháng (Flat Rate)
- Unlimited team members
- Giới hạn 500 responded conversations/tháng
- All Basic features

### Basic Plan - $29/agent/tháng ($26.10 annual)
**Channels & Inbox**
- Unlimited Email Inboxes
- Live Chat
- Social Media Channels (Facebook, Instagram, Twitter)
- Website Integration
- 1 Brand only

**Automation & Bots**
- Workflow Automation (Macros)
- Response Templates
- Chatbots (Hello Bot, Order Bot, FAQ Bot)
- Cues (Proactive Messages)
- Customer Intents detection

**Knowledge Base**
- Public FAQ
- Internal FAQ

**Reporting**
- Basic Reporting (conversations, messages)

### Pro Plan - $49/agent/tháng ($44.10 annual)
*Tất cả features Basic, plus:*

**Channels**
- SMS Channel
- Voice/VoIP Channel
- Unlimited Brands (Multi-store support)

**Customer Intelligence**
- Live View of Site Visitors
- Real-time browsing activity monitoring

**Customization**
- Custom Hosted Domain
- Customizable Email Appearance
- Status Page

**Reporting**
- Advanced Reporting
- Tags reporting
- Ecommerce integrations reporting
- Workflows reporting

### Plus Plan - $69/agent/tháng ($62.10 annual)
*Tất cả features Pro, plus:*

**Real-time Features**
- Peek (Live Screensharing/Co-browsing)
- In-Chat Video Calls
- Optional Classic Chat Mode

**Team Management**
- Departments (Staff Groups)
- Staff Performance Reporting
- Staff Shifts and Vacation Days
- Customizable Staff Roles & Permissions

**Customer Experience**
- Customer Satisfaction Ratings/Surveys
- FAQ Article Edit History

### Enterprise Plan - Custom Pricing
- White-glove service
- Custom solutions
- High-volume support
- Dedicated account management
- Custom conversation volume limits
- SLAs

### Volume-Based Plans
- **Starter Volume**: 500 conversations/tháng, unlimited staff
- **Volume Pro**: Custom conversation allowance + Pro features
- **Volume Plus**: Custom conversation allowance + Plus features

## 2. PHÂN LOẠI THEO QUY MÔ DOANH THU (GMV/Revenue)

### Starter/Hobby (<$10K/tháng)
**Recommended: Starter Plan ($59 flat)**
| Feature | Giá trị mang lại |
|---------|------------------|
| Unlimited team members | Flexibility khi scale |
| 500 conversations cap | Đủ cho volume thấp |
| Basic automation | Set-and-forget workflows |
| Simple chatbots | Giảm workload manual |

**Pain points addressed:**
- Price sensitive → Flat rate predictable
- Cần simplicity → Pre-built bots, không cần coding

### SMB ($10K-$100K/tháng)
**Recommended: Basic → Pro Plan**
| Feature | Giá trị mang lại |
|---------|------------------|
| Workflow Automation | Scale mà không tăng headcount |
| Cues (Proactive Messages) | Increase conversion rate |
| Multi-brand support (Pro) | Quản lý multiple stores |
| SMS/Voice channels | Omnichannel experience |
| Live View | Real-time engagement |

**Transition triggers từ Basic → Pro:**
- Mở thêm store/brand thứ 2
- Cần SMS support
- Muốn track visitor behavior

### Mid-market ($100K-$1M/tháng)
**Recommended: Plus Plan**
| Feature | Giá trị mang lại |
|---------|------------------|
| Peek (Live Screensharing) | Reduce resolution time |
| Video Calls | High-touch support |
| Departments | Organize by function |
| Staff Performance Reports | QA và training |
| CSAT Surveys | Measure effectiveness |
| Custom Roles | Security & compliance |

**Key requirements met:**
- Scalability: Department structure
- Integration: Deep Shopify/BigCommerce
- Dedicated support: Performance monitoring

### Enterprise (>$1M/tháng)
**Recommended: Enterprise Custom**
| Requirement | Solution |
|-------------|----------|
| Custom SLAs | Negotiated uptime |
| Account management | Dedicated CSM |
| Volume-based pricing | Cost optimization |
| Custom integrations | API + SDK access |
| HIPAA compliance | Available on Plus+ |

## 7. COMPETITIVE GAPS & LIMITATIONS

| Limitation | Impact |
|------------|--------|
| No free plan | Barrier for micro merchants |
| Limited search function | Hard to find old conversations |
| No vertical templates | More setup work needed |
