## 3. AI/AUTOMATION FEATURES (Deep Dive)

### Lyro AI Agent

**AI Model & Technology:**
- Powered by **Claude (Anthropic)** - not OpenAI/GPT
- Multi-agent architecture (multiple Claude instances working together)
- One agent rates conversation quality, another summarizes
- Fine-tuned for customer service use cases
- No hallucination by design - only uses provided knowledge base

**Knowledge Base Sources:**
- Website FAQ scraper (automatic)
- Help center article import
- Zendesk article import
- Manual Q&A upload
- URL content scraping
- Custom file uploads

**Resolution Rate Performance:**
| Metric | Value |
|--------|-------|
| Average resolution rate | 67% |
| Peak resolution rate | 90% |
| Email channel resolution | ~90% |
| Response time | 5-20 seconds |

**Language Support:**
- 48+ languages with native processing
- Not translation-based - actual understanding
- Auto-detect visitor language
- Respond in visitor's language

**Tone Customization:**
- Friendly (casual, warm)
- Neutral (balanced, professional)
- Formal (business, corporate)
- Custom instructions via Guidance rules

**Handoff Rules:**
- When Lyro doesn't know answer
- When visitor requests human
- After X messages without resolution
- For specific topics/keywords
- Time-based escalation

### Lyro Smart Actions (API Integrations)

**How Actions Work:**
1. Lyro recognizes customer intent requiring action
2. Collects necessary information via conversation
3. Executes API call to external system
4. Confirms completion to customer
5. Continues conversation or closes

**Pre-built Actions:**

| Category | Actions Available |
|----------|-------------------|
| **E-commerce** | Check order status, update shipping address, initiate return, check inventory, apply discount |
| **CRM** | Create lead, update contact, add note, assign to rep |
| **Scheduling** | Book appointment, check availability, send calendar invite |
| **Account** | Password reset, update preferences, change subscription |
| **Support** | Create ticket, escalate to department, send transcript |

**Supported Integrations:**
- Shopify (native, deep)
- WooCommerce
- Calendly

## 6. UNIQUE/DIFFERENTIATING FEATURES

### 1. Claude-Powered AI (vs. OpenAI competitors)
- Anthropic's Claude model
- Designed to be "helpful and harmless"
- Lower hallucination risk
- Higher trust for sensitive conversations

- Use Lyro with existing helpdesk

- Expands TAM beyond Tidio users

- Lyro responds to email tickets

- Differentiator vs. chat-only competitors

### 4. Free Copilot Chrome Extension
- Works with any helpdesk

- Competitive with paid tools

### 5. Guaranteed Resolution Rate (Premium)
- 50% AI resolution SLA

- Unique in SMB market

### 6. 48+ Native Languages
- Not translation-based
- True multi-lingual understanding

|----------|-------------------|-------------|
| FAQ automation | Lyro AI | Chatty (sales-focused) |

### When to Choose Tidio vs. Chatty

**Choose Tidio When:**
- Need AI for email ticketing channel
- Want to integrate with existing Zendesk/Intercom (Lyro Connect)
- Require enterprise features (SSO, compliance)
- Need multi-platform support beyond Shopify
- Have budget for Premium tier managed AI

**Choose Chatty When:**
- Shopify-only business
- Want simple, flat-rate pricing
- Prioritize sales conversion over support
- Budget-conscious (need AI under $50/mo)
- Value highest Shopify ratings/reviews
- Prefer integrated single product
