# Secondary Research Framework - Action Plan

This framework is like listen on the internet.
AI Agents, Skills, Workflow can apply to pull data, analysis automatically and weekly update

## 1. Product Intelligence

### Setup Actions
- [ ] Create competitor list (prioritize 5-7 direct competitors): Tidio, Vanchat
- [ ] Set up Claude code folder/files as database for tracking
- [ ] AI Agent analysis for weekly competitor monitoring
- [ ] Create Slack channel #chatty-competitive-updates for sharing updates

### Changelog & Release Monitoring
- [ ] Subscribe to competitor email newsletters -> AI agent listen email
- [ ] Follow competitor blogs via RSS (use Feedly) -> Setup AI agent to read
- [ ] Bookmark competitor changelog/release/What's new notes pages -> AI Agent pull changelog, release note weekly
- [ ] Set Google Alerts for "[competitor] + new feature/update/launch" -> RSS feed -> AI Agent can read

### App Store Tracking (Shopify-Specific)
- [ ] Screenshot competitor app listings monthly (description, pricing, features) -> AI command to pull
- [ ] Track review count and rating weekly
- [ ] Note any pricing tier changes or new plans -> Pricing change alert; Applisting & /pricing page
- [ ] Track their app categories and keyword positioning -> Bot crawl S Applisting

### Product Deep-Dive Actions
- [ ] Install competitor apps on test stores (create dedicated test stores)
- [ ] Document onboarding flow with screenshots/video
- [ ] Map feature set to yours (build comparison matrix)
- [ ] Test new features within 1 week of release -> human
- [ ] Note UX patterns and design choices

## 2. Positioning Intelligence

### Website Analysis Actions
- [ ] Screenshot competitor homepages quarterly (track messaging evolution)
- [ ] Document their headline, subheadline, and hero CTA
- [ ] List the customer segments they highlight
- [ ] Identify pain points they lead with
- [ ] Note social proof (logos, testimonials, metrics)

### Comparison Page Mining
- [ ] Find all competitor comparison pages (search "[competitor] vs [other]")
- [ ] Document how they position against others
- [ ] Extract their claimed differentiators
- [ ] Note which competitors they compare against (reveals who they fear)

### Messaging Framework Extraction
- [ ] Document their tagline and value proposition
- [ ] List 3-5 key benefits they emphasize
- [ ] Identify their ICP (ideal customer profile) from copy
- [ ] Note tone and voice (technical vs. friendly, enterprise vs. SMB)

### Case Study Analysis (Important)
- [ ] Collect all public case studies
- [ ] Extract: customer size, industry, use case, results claimed
- [ ] Identify which customer stories they promote most
- [ ] Note metrics they highlight (reveals what they optimize for)

### Ad Copy Research
- [ ] Use Facebook Ad Library to view active ads
- [ ] Search Google for competitor branded terms (see their ad copy)
- [ ] Document hooks and offers used
- [ ] Track promotional patterns (discounts, trials, demos)

### Positioning Map Exercise
- [ ] Define 2 axes most relevant to buyers
- [ ] Plot all competitors on the map
- [ ] Identify white space opportunities
- [ ] Update quarterly as market shifts
